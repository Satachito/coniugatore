# イタリア語動詞活用表 (Coniugazione dei verbi)

日本人学習者向けのイタリア語動詞活用表サイト。ビルド不要の静的サイトです。

姉妹サイト: [スペイン語動詞活用表](https://satachito.github.io/conjugador/)

## 使い方

`index.html` をブラウザで開くだけで動きます。ローカルサーバーを使う場合:

```sh
python3 -m http.server 8732
# → http://localhost:8732
```

## 機能

- **収録動詞 70語** — 不規則動詞・-isc型・つづり調整動詞・規則動詞に分類
- **収録時制** — 直説法(現在・近過去・半過去・遠過去・未来)、条件法現在、接続法(現在・半過去)、命令法(肯定・否定)、ジェルンディオ・過去分詞
- **近過去の助動詞表示** — essere 支配の動詞はバッジ表示し、過去分詞の性数一致 (andato/-a/-i/-e) も表に反映
- **不規則箇所のハイライト** — 規則活用と異なる形を赤字で表示
- **検索** — イタリア語(アクセント記号なしでも可)・日本語の意味の両方で検索
- **練習モード** — ランダム出題のクイズ。アクセント記号の入力ボタン付き

## 動詞の追加方法

`app.js` の `VERBS` 配列にエントリを追加します。規則動詞なら `{ inf: "cantare", ja: "歌う" }` だけで全時制が自動生成されます。不規則な場合は以下のフィールドで指定します:

- `presOv` — 直説法現在の6人称配列での上書き(接続法現在もここから自動派生)
- `prStem: "pres"` — 遠過去の強変化語幹 (presi, prese, presero 型)
- `wkStem: "fac"` / `wkCls: "ere"` — 弱語幹(fare→facevo, facendo, facessi などに使用)
- `futS: "andr"` — 未来・条件法の語幹
- `isc: true` — capire 型の -isc- 挿入
- `aux: "e"` — 近過去が essere 支配(過去分詞が性数一致)
- `part` / `tu` / `impVoi` — 不規則な過去分詞・命令形

-care/-gare の h 挿入、-ciare/-giare/-iare の i 調整は自動で処理されます。追加後は `node test.js` で既存の活用が壊れていないことを確認してください。
